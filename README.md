# 🌍 Climate Nexus – A One-Stop Platform for Climate Action  

![Hack the Future](https://github.com/ShahxHussain/VizAI_Data_Visualisation_Assistant/blob/main/Extras/Hack%20the%20future.png)

## 🌟 Inspiration  

Climate change is **one of the most pressing challenges of our time**, affecting millions of people globally. From **climate refugees to marine pollution, deforestation, and food insecurity**, the need for **technological solutions** is greater than ever. Climate Nexus was born out of a desire to **bridge the gap between awareness and action** by **leveraging AI, data science, and community engagement** to drive meaningful climate solutions.  

---

## 🚨 The Problem  

Despite global efforts, climate action remains **fragmented, inaccessible, and underutilized**. Some key issues include:  

- 🌊 **Lack of real-time data on ocean pollution** affecting marine biodiversity.  
- 🏠 **Climate refugees struggle to find aid and relocation support**.  
- 📉 **Limited awareness about technological solutions** for climate change.  
- 🔍 **Governments and NGOs lack centralized climate data** for policy-making.  

---

## 💡 The Solution  

**Climate Nexus** is a comprehensive, technology-driven platform that empowers **activists, researchers, policymakers, and individuals** to take real climate action. It offers:  

### 🔹 **Climate Action Hub Center**  
📚 A knowledge hub covering **movements, policies, and innovative climate solutions**.  
📊 Open-access **data, research papers, and case studies** to inspire action.  

### 🔹 **Tracking & Aiding Climate Refugees**  
📍 **Visualizing climate refugee journeys** with **interactive NASA-powered maps**.  
🤝 A **community-driven aid network** connecting displaced populations with resources.  

### 🔹 **Climate Assistant (AI Chatbot)**  
🤖 AI-driven chatbot providing **real-time insights on climate solutions, sustainability tips, and policy recommendations**.  

### 🔹 **AquaGuardAI – Plastic Detection in Deep Seas**  
🌊 AI-powered **underwater drones** detecting and tracking **plastic pollution**.  
📈 Real-time **pollution heatmaps** and predictive analytics to prevent ocean contamination.

## 📝 Installation and Setup Guide  
## **Step 1: Prerequisites**  

Ensure you have the following installed on your system:  

- **Node.js**  
- **Git** (for cloning the repository)  
---

## **Step 2: Clone the Repository**  

Clone the project repository from GitHub:  
```bash
git clone https://github.com/ShahxHussain/Climate-ActionHub-and-Assistance
cd Climate-ActionHub-and-Assistance
```

## **Step 3: Install Dependencies**

```bash
npm install
```

## **Step 4: Run code locally**

```bash
npm start
```

## **Step 5: Get Together.ai for ClimateAssistant V1.0]**

- Get Together api [here](https://api.together.ai/)
1. Sigup
2. Proceed To Dashboard
3. Get your api key at keys section
